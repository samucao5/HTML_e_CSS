quais os tipos de dados na programação

os tipos de dados sao como uma variavel vai se identificar no codigo

criador:Samuel Felipe Candido Monteiro Da Silva

algumas linguagens de programacao demandam que a variavel tenha seu tipo especificado como por exemplo o c++

abaixo alguns tipos de dados

  • inteiro
  • float
  • char
  • string

INTEIRO

os valores inteiros se tratam de numeros sem ponto flutuantes

ex: 1, 6, 2000, -600, etc

para se declarar devemos colocar desta forma ex:int nome_da_variavel = 5;

FLOAT

os valores float sao tanto valores inteiros como valores de ponto flutuante
ex: 51.53, 8, -234.6, -66

para se declarar devemos colocar desta forma ex:float nome_da_variavel = 4.46;

CHAR

o char e um caractere do teclado pode ser numero, letra, caractere especial
ex: a, 5, 6.34, -1, &

OBS: nao pode ser usado para fazer soma, ele so aparece na tela

para declarar e necessario colocar desta forma ex: char nome_da_variavel = 's';

STRING

a string e o conjunto de char ao qual formam palavras
ex: bom, 54, 8.6, &, etc

para declarar e necessario colocar desta forma ex: std::string nome_da_variavel = "bom";
